# AWS VPC Terraform module
# Usage (example)

```
   public_cidrs  = ["10.0.1.0/24", "10.0.2.0/24"]
   private_cidrs = ["10.0.3.0/24", "10.0.4.0/24"]
```
subnets is created depending on the number of cidr-blocks
